# What the project does?

The AI Research Assistant is a platform that acts as an inteliggent assitant for scientific research. The system analyzes academic papers in the field of Artificial Intelligence, performs exploratory data analysis, classifies documents using Machine Learning, and allows users to ask questions about the papers using LLMs with Retrieval-Augmented Generation (RAG).
